# 03 · 상품 상세 (`/products/$id`)

> 와이어프레임의 좌측 이미지 / 우측 정보 구조 유지 — 트렌드(이미지 우선·미니멀 정보)에 맞춰 큰 갤러리 + 정보 + 드롭 이력.

## 구성 → 구현
| 영역 | 구현 |
|---|---|
| 이미지 갤러리 | `ImageGallery`(메인 4:5 + 썸네일 스트립) — `buildGallery(thumbnailKey)` |
| 상품 정보 | 카테고리·이름(serif)·가격(가격 미정)·설명·등록일 |
| 드롭 이력 | `useDropList()` → `productId` 필터 → 각 드롭 상세 링크 |

## 데이터 · API
| 액션 | 경로 | 상태 |
|---|---|---|
| 상품 단건 | `GET /api/v1/products/{id}` → `ProductResponse` | ✅ 실계약(apiFetch + Zod) |

- 상품은 직접 구매 불가(드롭으로만) → 구매는 드롭 이력의 드롭 상세에서 진입.
- 검증: 빌드 + 스크린샷. 카드(목록/홈)→상세 링크 연결.
